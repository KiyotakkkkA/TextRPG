# Обновление v2.0.1 Alpha
Дата: 20.03.2025 04:55:03
Сборка: 20250201

## Изменения
- Добавлена система журнала изменений (Changelog)
- Реализована поддержка ASCII-арта для названий обновлений
- Добавлен новый экран для просмотра информации об обновлениях
- Добавлена команда `python pymust update --log --name=ИмяОбновления`

## Новые возможности
- ASCII-арт автоматически генерируется из названия обновления
- Арт отображается в главном меню 
- Новый пункт меню для просмотра деталей обновления
- Интеграция с системой версионирования

## Улучшения интерфейса
- Добавлен яркий и заметный пункт меню для новых обновлений
- Информация об обновлении отображается в удобном формате
- Подсветка заголовков в тексте обновления
